Projeto desenvolvido com Spark Framework e Angular

About project:

O projeto envolve uma aplicação para gerenciamento de escolas, com funcionalidades como cadastro, visualização, atualização e exclusão de escolas, turmas, alunos e professores. Utiliza uma arquitetura de API RESTful para comunicar o frontend (Angular) com o backend (Spark Java), permitindo a troca de dados de forma eficiente e padronizada.
